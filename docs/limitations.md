# Limitations

White Pony is a research project. The public repo should be read as process evidence, not as a performance claim.

## Main Limits

- Historical results may not persist.
- Backtests can change with data revisions, universe definitions, and evaluation windows.
- Strong results can come from multiple testing.
- Model retraining and data refreshes can materially change results.
- Cost and slippage assumptions are incomplete until explicitly tested.
- Simulated execution can differ from real fills.
- Delayed logs are still early evidence.

## Publication Boundary

The public repository omits private formulas, model parameters, and execution logic. That protects IP and security, but it also means the public examples cannot reproduce the private system.

## Not A

- trading signal service;
- full strategy disclosure;
- performance guarantee;
- investment recommendation.
