# Risk Controls

This note lists the kinds of risk checks that can be discussed publicly.

## Research Checks

A result should be treated as tentative until it has been checked for:

- position concentration limits;
- turnover;
- drawdown;
- cost sensitivity;
- lag sensitivity;
- subperiod checks;
- correlation and redundancy checks;
- multiple-testing risk.

## Public Boundary

This repo does not include broker code, live execution scripts, or operational controls, etc.

It can be used for high-level ideas such as:

- broker separation from research code;
- delayed aggregate reporting;
- not publishing current holdings or order details;
- using manual review before adding charts or papers.
