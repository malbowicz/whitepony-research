# Charts

This folder is for reviewed public charts.

No private charts have been copied into this public repository yet. Charts should be added only after manual review.

## Before Adding A Chart

- remove private paths from captions, legends, metadata, and filenames;
- check that it does not reveal current holdings, targets, or account details;
- label the chart as historical, simulated, paper, or delayed aggregate;
- keep the caption specific and cautious.
