## Public Release Checklist (for me)

Before adding a PDF, review it for any of these:
- private file paths
- proprietary formulas, thresholds, or feature definitions
- current holdings, target weights, broker/account details, or order logs
- language that overstates historical or live results
- missing limitations or risk disclosures
- anything else that might be bad to post publicly

Historical, backtested, and early live results HAVE TO BE labeled as research evidence, and NOT AS proof of future performance.
